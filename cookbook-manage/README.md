# cookbook-manage

> 菜谱大全后台管理系统， 本系统基于[vue-element-admin](https://panjiachen.github.io/vue-element-admin-site/zh/) 开发，  更多组件使用信息请[查看](http://element-cn.eleme.io/#/zh-CN/component/installation) 

线上地址



## 如何使用

```bash
# 安装依赖
npm install

# 建议不要用cnpm  安装有各种诡异的bug 可以通过如下操作解决npm速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 本地开发 启动项目
npm run dev

# 开发环境 打包
npm run build

# 国内在安装node-sass的时候可能会安装不上，使用以下方式即可解决
npm install -g mirror-config-china --registry=http://registry.npm.taobao.org
npm install node-sass

```

## 网站实现流程

该后台路由动态生成，简述下登录和权限验证的实现思路：

- **登录：**当用户登录成功后，服务端会返回一个**token**，拿到token之后（将这个token存贮到cookie中，保证刷新页面后能记住用户登录状态），前端会根据token再去拉取一个 **user_info** 的接口来获取用户的详细信息（含用户权限、用户名、头像等信息）。
